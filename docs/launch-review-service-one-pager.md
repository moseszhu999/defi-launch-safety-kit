# Launch-Readiness Review Service

## Positioning

A short, defensive pre-audit preparation review for small DeFi/token teams preparing for formal audit, testnet, or mainnet launch.

This is not a formal security audit. It is a structured readiness pass to help a team understand owner/admin permissions, launch controls, deployment readiness, and documentation gaps before a full audit or launch decision.

## Typical scope

- Run DeFi Launch Safety Kit on the repository
- Generate Markdown, HTML, JSON, and SARIF reports
- Review owner/admin and privileged function surface
- Review mint, burn, pause, fee/tax, transfer-control, rescue, withdraw, and upgrade paths
- Review Foundry/Hardhat readiness, tests, deployment scripts, CI, and coverage signals
- Generate an audit-preparation pack
- Provide a concise remediation checklist

## Deliverables

```text
1. Technical risk report
2. Owner/admin permission review
3. Contract surface map
4. Deployment checklist
5. Known-risk notes
6. Questions for the project team
7. Project readiness summary
```

## Best-fit projects

- ERC20 launches
- Staking contracts
- Vesting contracts
- Airdrop contracts
- Small DeFi teams preparing for audit or mainnet launch
- Hackathon or grant-funded teams that need a first structured review

## Not included

- No guarantee of safety
- No replacement for formal audit
- No financial advice
- No market-making, token promotion, or investment advice
